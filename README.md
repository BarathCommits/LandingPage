# PatentChain.ai Landing Page

A modern, responsive landing page for PatentChain.ai - a startup company focused on democratizing inventions through AI-powered patent creation and blockchain-based licensing.

## 🚀 Vision

PatentChain.ai aims to revolutionize the patent industry by:
- **Democratizing Inventions**: Making patent creation accessible to inventors, researchers, students, professionals, engineers, and doctors without expensive attorney fees
- **AI-Powered Patent Creation**: Using artificial intelligence to generate comprehensive patent applications
- **Blockchain Tokenization**: Converting existing patents into tradeable tokens on the blockchain
- **Global Licensing Platform**: Enabling worldwide licensing and monetization of patented technologies

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. The landing page will load with all functionality

### Development
To modify the landing page:

1. **Content Changes**: Edit `index.html` for text, links, and structure
2. **Styling Changes**: Modify `styles.css` for colors, layout, and design
3. **Functionality Changes**: Update `script.js` for interactive features
